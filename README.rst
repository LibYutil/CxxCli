
CxxCli
======
C++11 Cli template library

Trivialises cli command parsing logic

Tested on MSVC 19

Usage
-----
See tests

CMake
*****
- Include ``./include`` path
  - ``target_include_directories(<target> <PRIVATE | PUBLIC | ...> CxxCli/include)``
- ``#include "CxxCli/CxxCli.hpp"``
